// Embedded Systems Engineer & Hardware Hacker

EmbedLab is a curated collection of real-world embedded systems projects built across Arduino, ESP32, STM32 and Raspberry Pi platforms. Every project includes a full component BOM, SVG circuit diagram, and C/C++ source code hosted on GitHub.

The goal is to bridge the gap between theory and hardware — each project is documented at the same standard you would expect from a professional engineering team. From a simple Arduino blinky to a sensorless BLDC motor controller, every detail is covered.

// GETTING STARTED

Sign in, browse a project family, open a card to see the full component BOM and circuit diagram, then clone the GitHub repo and flash your board.

$ git clone https://github.com/embedlab/[project-name] $ cd [project-name] $ make $ make flash BOARD=arduino_uno # or esp32, stm32f4, etc.
